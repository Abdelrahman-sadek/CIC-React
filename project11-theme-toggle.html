📘 مشروع 11: Theme Toggle App (Dark/Light)

🎯 هتتعلم إيه في المشروع ده؟

  • إزاي تبدل بين الوضع الليلي والوضع العادي في React
  • تستخدم useState عشان تدير حالة الثيم
  • تغير الستايل حسب الحالة
  • تخلي الواجهة تفاعلية أكتر وجذابة
🛠️ خطوات إعداد المشروع
  1. 1اعمل مشروع جديد باسم theme-toggle-app بنفس الطريقة اللي اتعلمناها:
    2. Terminal 
    npm create vite@latest theme-toggle-app -- --template react
  2. 2ادخل على فولدر المشروع وثبت الحاجات المطلوبة:
    3. Terminal 
    cd theme-toggle-app
npm install
npm run dev
💡 افتح http://localhost:5173 في المتصفح وشوف التطبيق.

🧠 المفاهيم الأساسية
  • useState: لإدارة حالة الثيم (داكن/فاتح).
  • Conditional CSS: تغير الكلاس أو الستايل حسب الحالة.
  • تجربة المستخدم: خلي الواجهة تفاعلية وسهلة للمستخدم.
