📘 مشروع 4: useState Counter

🎯 هتتعلم إيه في المشروع ده؟

  • إزاي تستخدم useState في React
  • تتعامل مع الأحداث (events) زي الضغط على زرار
  • تعمل عداد (Counter) بسيط
  • تحدث الواجهة لما الstate تتغير
🛠️ خطوات إعداد المشروع
  1. 1اعمل مشروع جديد باسم usestate-counter بنفس الطريقة اللي اتعلمناها:
    2. Terminal 
    npm create vite@latest usestate-counter -- --template react
  2. 2ادخل على فولدر المشروع وثبت الحاجات المطلوبة:
    3. Terminal 
    cd usestate-counter
npm install
npm run dev
💡 افتح http://localhost:5173 في المتصفح وشوف التطبيق.

🧠 المفاهيم الأساسية
  • useState: هو hook بيديك متغير state ودالة تغيره.
  • Event Handling: يعني تتعامل مع الأحداث زي الضغط على زرار.
  • re-render: كل ما الstate تتغير، React بيعيد رسم المكون تلقائي.

💬 شوية نصايح / أحسن ممارسات
  • جرب تزود أكتر من عداد في نفس المكون.
  • خلي كل زرار ليه وظيفة واضحة.
  • استخدم useState دايمًا مع القيم اللي بتتغير في الواجهة.
