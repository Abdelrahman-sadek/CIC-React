🧠 المفاهيم الأساسية

ما هي المكونات (Components)؟

المكونات هي قطع من الكود قابلة لإعادة الاستخدام تعيد JSX. يمكن كتابتها كدوال:

// مكون بسيط
function Welcome() {
  return <h1>مرحباً بك في React!</h1>;
}

// استخدام المكون
function App() {
  return (
    <div>
      <Welcome />
    </div>
  );
}
ما هي Props؟

Props هي وسيلة لتمرير البيانات من مكون أب إلى مكون فرعي. مثال:

// مكون يأخذ props
function Welcome(props) {
  return <h1>مرحباً، {props.name}!</h1>;
}

// تمرير البيانات عبر props
function App() {
  return (
    <div>
      <Welcome name="أحمد" />
      <Welcome name="فاطمة" />
    </div>
  );
}
مهم: Props تكون للقراءة فقط ولا يمكن تغييرها داخل المكون
إعادة الاستخدام (Reusability)

المكونات تسمح لك بإعادة استخدام نفس الكود مع بيانات مختلفة:

// نفس المكون مع بيانات مختلفة
<Welcome name="أحمد" />
<Welcome name="فاطمة" />
<Welcome name="عمر" />

// النتيجة:
// مرحباً، أحمد!
// مرحباً، فاطمة!
// مرحباً، عمر!

✍️ التطبيق خطوة بخطوة

الخطوة 1: إنشاء مكون Welcome.jsx

أنشئ ملف جديد في src/Welcome.jsx واكتب الكود التالي:

// مكون Welcome يأخذ props
function Welcome(props) {
  return <h1>Welcome, {props.name}</h1>;
}

// تصدير المكون ليتم استخدامه في ملفات أخرى
export default Welcome;
شرح الكود:
  • props هو كائن يحتوي على البيانات المرسلة للمكون
  • props.name نصل للقيمة المرسلة باسم "name"
  • export default نصدّر المكون ليتم استيراده في ملفات أخرى
الخطوة 2: تعديل App.jsx

افتح src/App.jsx واستبدل محتواه بالكود التالي:

import Welcome from './Welcome';

function App() {
  return (
    <div>
      <Welcome name="Fatma" />
      <Welcome name="Omar" />
      <Welcome name="Sara" />
    </div>
  );
}

export default App;
نقاط مهمة في الكود:
  • import Welcome نستورد المكون من ملفه
  • name="Fatma" نمرر البيانات عبر props
  • نستخدم نفس المكون مع بيانات مختلفة
الخطوة 3: تأكد من main.jsx

تأكد أن ملف src/main.jsx يحتوي على الكود التالي:

import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

🧪 الكود النهائي

src/Welcome.jsx
function Welcome(props) {
  return <h1>Welcome, {props.name}</h1>;
}

export default Welcome;
src/App.jsx
import Welcome from './Welcome';

function App() {
  return (
    <div>
      <Welcome name="Fatma" />
      <Welcome name="Omar" />
      <Welcome name="Sara" />
    </div>
  );
}

export default App;
src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

💬 نصائح إضافية / أفضل الممارسات

  • استخدم أسماء واضحة للـ props لتسهيل فهم الكود.
  • يمكنك تمرير أي نوع من البيانات عبر props (نصوص، أرقام، مصفوفات، كائنات).
  • استخدم destructuring لاستخراج props: function Welcome({name}).
  • جرب إضافة props جديدة مثل age أو city.
  • حافظ على المكونات بسيطة وقابلة لإعادة الاستخدام.
