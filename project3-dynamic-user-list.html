🧠 المفاهيم الأساسية

ما هي map؟

map هي دالة في JavaScript تُستخدم لتحويل كل عنصر في مصفوفة إلى عنصر جديد. في React نستخدمها لعرض ليستة من العناصر:

// مصفوفة المستخدمين
const users = [
  { id: 1, name: 'Omar' },
  { id: 2, name: 'Fatma' },
  { id: 3, name: 'Sara' }
];

// استخدام map لعرض كل مستخدم
users.map(user => );
ما هي خاصية key؟

key هي خاصية خاصة في React تساعد في تتبع العناصر عند تحديث القائمة. يجب أن تكون قيمة فريدة لكل عنصر:

{users.map(user => )}
مهم: لا تستخدم index كمفتاح إلا إذا لم يكن لديك id فريد
ما هو UserCard؟

UserCard هو مكون React يعرض بيانات مستخدم واحد. يمكنك تمرير البيانات له عبر props:

// مكون UserCard
function UserCard(props) {
  return 
Welcome, {props.name}
;
}

الخطوة 1: إنشاء مكون UserCard.jsx

أنشئ ملف جديد في src/UserCard.jsx واكتب الكود التالي:

// مكون UserCard
function UserCard(props) {
  return 
Welcome, {props.name}
;
}

export default UserCard;
شرح الكود:
  • المكون يأخذ props ويعرض اسم المستخدم
  • يمكنك إضافة خصائص أخرى مثل العمر أو المدينة
