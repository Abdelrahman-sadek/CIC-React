🛠️ إعداد المشروع

الخطوة 1: إنشاء مشروع React جديد

افتح Terminal أو Command Prompt واكتب الأوامر التالية:

# إنشاء مشروع جديد باستخدام Vite
npm create vite@latest hello-react -- --template react

# الانتقال إلى مجلد المشروع
cd hello-react

# تثبيت المكتبات المطلوبة
npm install

# تشغيل المشروع
npm run dev
ملاحظة: Vite هو أداة بناء سريعة وحديثة لمشاريع React

🧠 المفاهيم الأساسية

ما هو React؟

React هو مكتبة JavaScript تم تطويرها من قبل Facebook لبناء واجهات المستخدم التفاعلية. يتميز React بـ:

  • Component-Based: يقسم الواجهة إلى مكونات صغيرة قابلة لإعادة الاستخدام
  • Virtual DOM: يحسن الأداء عبر تحديث أجزاء محددة من الصفحة فقط
  • Declarative: تصف كيف تريد أن تبدو الواجهة وليس كيفية تحديثها
ما هو JSX؟

JSX هو امتداد لـ JavaScript يسمح لك بكتابة HTML داخل JavaScript. مثال:

// JSX - يبدو مثل HTML لكنه JavaScript
const element = <h1>مرحباً بالعالم!</h1>;

// يتم تحويله إلى JavaScript عادي
const element = React.createElement('h1', null, 'مرحباً بالعالم!');
مهم: في JSX، استخدم className بدلاً من class و htmlFor بدلاً من for
ما هي المكونات (Components)؟

المكونات هي قطع من الكود قابلة لإعادة الاستخدام تعيد JSX. يمكن كتابتها كدوال:

// مكون بسيط
function Welcome() {
  return <h1>مرحباً بك في React!</h1>;
}

// استخدام المكون
function App() {
  return (
    <div>
      <Welcome />
    </div>
  );
}
ما هو ReactDOM؟

ReactDOM هو المكتبة المسؤولة عن عرض مكونات React في DOM الحقيقي:

import ReactDOM from 'react-dom/client';
import App from './App';

// إنشاء root وعرض التطبيق
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);

✍️ التطبيق خطوة بخطوة

الخطوة 1: تعديل main.jsx

هذا الملف هو نقطة البداية لتطبيق React. افتح src/main.jsx وعدله كما يلي:

import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.jsx'
import './index.css'

// إنشاء root element - هذا هو المكان الذي سيتم عرض التطبيق فيه
const root = ReactDOM.createRoot(document.getElementById('root'))

// عرض مكون App الرئيسي
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
)
شرح الكود:
  • React.StrictMode يساعد في اكتشاف المشاكل أثناء التطوير
  • createRoot ينشئ نقطة عرض للتطبيق
  • render يعرض المكون في الصفحة
الخطوة 2: إنشاء مكون App.jsx

هذا هو المكون الرئيسي لتطبيقنا. افتح src/App.jsx واستبدل محتواه بـ:

import React from 'react'
import './App.css'

// المكون الرئيسي للتطبيق
function App() {
  return (
    <div>
      <h1>Hello React!</h1>
      <p>Welcome to your first React app 🚀</p>
    </div>
  );
}

export default App;
نقاط مهمة في الكود:
  • استخدام className بدلاً من class
  • تصدير المكون باستخدام export default
  • هذا هو أبسط شكل لمكون React

🧪 الكود النهائي

src/App.jsx
function App() {
  return (
    <div>
      <h1>Hello React!</h1>
      <p>Welcome to your first React app 🚀</p>
    </div>
  );
}

export default App;
src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
